# Magic Rule

科学上网配置文件和规则集仓库

## 目录结构

```
.
├── clash/          # Clash 配置文件
├── rules/          # 分流规则集
└── README.md       # 说明文档
```

## 使用说明

### Clash 配置

将配置文件放在 `clash/` 目录下，可以根据不同的使用场景创建多个配置文件。

### 分流规则

自定义规则集放在 `rules/` 目录下，支持：
- 域名规则
- IP-CIDR 规则
- GEOIP 规则

## 注意事项

- ⚠️ 不要提交包含敏感信息（如节点信息、密码）的配置文件
- 建议将个人配置文件命名为 `*-private.yaml`，已在 `.gitignore` 中排除
